# Superseded plans

These plans were written under the [2026-04 program review](../../PROGRAM-REVIEW-2026-04.md)
when Vibe Dash was heading toward a broader platform (intelligence, git sync,
team mode). The mid-2026 **Phase 1A–1D cuts** removed those feature areas to
refocus on a local-first **Dashboard + Kanban + MCP** for a solo user, so the
plans here describe work that was cut or never built.

They are kept for historical context — not as roadmap. For current direction see
the [Roadmap](../../../README.md#roadmap).

| Plan | Why superseded |
|------|----------------|
| [R12.1 — NL query + digests](R12.1-nl-query-and-digests.md) | Intelligence layer cut in Phase 1C |
| [R12.2 — Git host sync](R12.2-git-host-sync.md) | Git ingestion infra removed in Phase 1B; never built |
| [R13.1 — Self-hostable team mode](R13.1-self-hostable-team-mode.md) | Auth / team mode / `users` removed in Phase 1C |

Still-relevant plans (partially shipped) remain under [docs/plans/](../../plans/).
